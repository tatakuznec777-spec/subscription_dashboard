
## Запуск

### Локально
```
npm install
npm run dev

```

Откройте http://localhost:3000


### Docker

`docker-compose up --build`

### API


Приложение использует Mock API: https://gitverse.ru/semao0/mock_api_irg

Для работы необходимо подключение к интернету.


# Мотивация участия в проекте, для которого выполенно это тестовое задание:

### 1. Что тебя привлекает в этом проекте? 

```
Мне нравится, что это не абстрактный UI‑набор, а часть AI‑платформы: интерфейс здесь работает в связке с AI‑агентом, который находит подписки и присылает события через SSE. Для меня важно, чтобы фронтенд не просто отображал данные, а грамотно раскрывал возможности бэкенда и AI.

В этом кейсе как раз видно, как хороший интерфейс превращает «сырые» данные в понятные действия для пользователя (отмена, оплата, фильтрация).

Хочу поучаствовать в создании продукта, где ИИ и UX работают вместе и приносят реальную пользу.

```

### 2. Как ты видишь свою роль в команде?

```

Моя роль — сделать так, чтобы пользователь не думал о том, как устроена система, а просто решал свои задачи: увидел срочный платёж, нажал «Отменить», и всё сработало предсказуемо.

Я возьму на себя фронтенд‑реализацию пользовательских историй.

Также готова помогать с документацией и демо‑версией, чтобы команде было проще показывать прогресс.

```

### 3. Сколько часов в неделю готова уделять и на какой срок?

```

Могу стабильно уделять 20-25 часов в неделю и выше, если появятся срочные задачи или нужно будет закрыть важный этап. Готова работать по проекту такой срок, который потребуется для решения поставленных задач, даже если это будет дольше, чем было изначально заявлено(1,5-2 месяца).

```
